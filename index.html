CSS Cheat Sheet

Selectors

Selector Example Description
class .country Selects all elements with a class of Country
#id #france Selects the element that has the id of France
* * Selects all elements
element h2 Selects all <h2> elements
element.class h2.country Selects all <h2>elements with a class of Country
element, element h1, div Selects all <h1> and all <div> elements
element element span p Selects all <span> elements inside <p> elements
element>element span > p Selects all the <p> elements where the parent is a <span> element
element + element div + p Selects all the <p> elements that are placed immediately after <div> elements
element1~element2 p ~ div Selects every <div> element that are preceded by a <p> element
